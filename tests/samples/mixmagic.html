What is MixMagic:

MixMagic is harddrive sound mixing program for Gnome, it can handle large (larger then system memory) samples. MixMagic is able to mix as many waves as your CPU can handle.

What does it look like:

Click here for a screenshot of MixMagic.

Features:

  • Unlimited number of tracks.
  • Unlimited size of samples.
  • "Smart" memory usage of samples.
  • A nice Gnome GUI.
  • Loopblock.
  • Multi threaded.
  • Imports .wav files.

Limitations:

  • Only 16 bits stereo .wav files with the rate 44100 are currently supported.

Todo list:

  • Recording support.
  • Support for mono samples.
  • Volume and Pan envelope editing.
  • Effect, input and output plugins.
  • Internationalization.
  • And more...

Requirements:

  • Gnome 1.0 or greater.
