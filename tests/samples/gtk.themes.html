GTK Themes Supported By KDE2 - gtk.t.o News
Posted by ovadose on 26th July 2000 @ 10:33 PM PST
As i'm sure most of the avid slashdot fans out there are already aware, there is now support for pixmap themes (our most populated catagory) in KDE2 beta's thanks to an added feature in QT. The announcement also claims the themes run faster in KDE2 than under native GTK which sounds interesting. For pretty screenshots go have a look at the announcement. More integration of formats across the unix world can only be a good thing.

GTK Theme Switch 0.3 Released - gtk.t.o News
Posted by ovadose on 20th July 2000 @ 08:55 PM PST
GTK Theme Switch is a program for switching themes in the same manner as the Gnome Control Center however it is designed to do this from the command line. The program provides most of the same functions as Gnome CC as well as being dockable and comes with a simple GUI. If you're interested you can check it out at http://www.muhri.net/gthemeswitch.php3.

GTK+ and GLib 1.3.1 Released!!! - gtk.t.o News
Posted by adraken on 14th July 2000 @ 10:58 PM PST
