XML for SCRIPT 0.2
Michael Houghton <mike at idle.org> - July 15th 2000, 20:28 EST

XML for SCRIPT is a simple, fast, non-validating XML parser, written in JavaScript. It is suitable for experimental or educational purposes, or whatever other use you can put it to. The parser has a trivial document object model, and should run happily in a version 3 or 4 browser; a demonstration application is included online. XML for SCRIPT lacks some well-formedness constraints, and has no support whatsoever for entity declarations and the DOCTYPE tag, but should be useful for simple well-formed XML documents.

Changes: Switch to a single-pass parser, reworked DOM methods, and silent ignoring of processor instructions, entity declarations, and DTDs rather than failing.
